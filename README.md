# Spacecapades!

Spacecapades! is an infinite scroller game made using HTML Canvas, CSS, and JavaScript

### Works On

Firefox, Safari, DuckDuckGo, Internet Explorer 

>[!NOTE]
>Does not work on Chromium-based browsers, (e.g. Brave, Chrome, Edge, Opera)
